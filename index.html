CSS selectors and specificity

In a CSS file a selector is the first part of a CSS Rule. It is a structured element with different terms that tell the browser which HTML element you want to select and what kinf of CSS property values inside the rule should be applied to them. The selected element are known as the subject of the selector.

Selector Name Description
* Universal Selector targets any element
h1 Type selector targets any element of type <h1>
.special Class selector targets all class="special" elements inside a HTML element
h1.special Class selector targets all class="special" elements inside a <h1> element
h1.special.name Class selector targets all class="special" AND class="name" elements inside a <h1> element<
#food ID selector tarets any element with the id="food"
a[title] Attribute selector targets elements based on the presence of the attribute 'title' on an element
a[href="https://google.com"] Attribute selector targets based on the presence of an attribute with this value
a:hover Pseudo-class selector the target 'a' is styled by the 'hover' value
p::first-line Pseudo-element selector selects a certain part of an element and styles it
